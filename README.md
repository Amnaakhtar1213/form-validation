## FORM-VALIDATOR

A simple registered form that let the user to enter username, email, password, and confirm password.The form show error if one of the input is invalid or empty.Error represented in red outline ans success represented in green outline.

-----

##  FEATURES

- Enter Username (more than 4 char)
- Enter Email (valid email)
- Enter password (more than 6 char, less than 25 char)
- Enter confirm password (same as password)
- show Error (if error detect)
- show success (when registration complete)

----

## TECHNOLOGY

1. **HTML**
2. **CSS**
3. **JAVASCRIPT**
